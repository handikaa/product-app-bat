# product_bat

Flutter version using : 3.32.5
Dart version using : 3.8.1

## Getting Started
## App Description
Aplikasi sederhana untuk simulasi toko online

## Feature
Fitur : 
1. List Produk
2. Cari produk
3. Lihat halaman detail produk
4. Keranjang (tambah, hapus, edit)
5. cara hapus produk di keranjang dengan swipe produk ke kiri (seperti pada UI preview)

## How to run the project
1. clone github https://github.com/handikaa/product-app-bat.git di IDE
2. set folder project di folder yang ditentukan
3. buka folder project product-app-bat di IDE
4. jalankan "flutter run" di terminal, atau run debugging langsung jika menggunakan VS Code
