## How to run the project
1. clone github https://github.com/handikaa/product-app-bat.git di IDE
2. set folder project di folder yang ditentukan
3. buka folder project product-app-bat di IDE
4. jalankan "flutter run" di terminal, atau run debugging langsung jika menggunakan VS Code
